## Amplification

| Parameter | Value |
|-----------|-------|
| Amplifier | Behringer NX6000D |
| Mode | Stereo (1 channel per cabinet) |
| Load | 8Ω (2 drivers in series per cabinet) |
| Power @ 8Ω | ~800W real-world (per channel) |

### Wiring (per cabinet)

```
NX6000D Channel ──┬── Driver 1 (+) ── Driver 1 (-) ──┬── Driver 2 (+) ── Driver 2 (-) ── Return
                  │                                   │
              (series connection)                 (continues)
```

Voice coils wired in series: 2Ω + 2Ω = 4Ω per driver  
Two drivers in series: 4Ω + 4Ω = 8Ω per cabinet
