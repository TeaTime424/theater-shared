## Assembly Sequence (Baffle-Last)

### Phase 1: Preparation
1. Laminate baffle layers (glue + clamp 24hrs)
2. Cut driver holes in laminated baffle (router circle jig, Ø16.875")
3. Build window brace frames (glue rails + gussets)

### Phase 2: Box Assembly
4. Assemble box: Back → Sides → Top/Bottom (glue + clamp)
5. Install cleats on inside faces, 2" from front edge (glue + screw)

### Phase 3: Internal Bracing
6. Install front-to-back window braces (glue to back panel and cleats)
7. Install side-to-side window braces (glue to side panels)

### Phase 4: Wiring & Damping
8. Install terminal cup in back panel
9. Run speaker wire from terminal to driver location
10. Install damping material on internal surfaces

### Phase 5: Baffle Installation
11. Apply gasket tape or silicone to cleat faces
12. Install baffle (screw into cleats from front)
13. Mount drivers to baffle (T-nuts pre-installed)
14. Connect speaker wire to drivers
