## Wiring — 8Ω Series

```
NX6000D
┌──────┐
│ Ch A ├────► Cab 1 (4Ω) ──jumper──► Cab 2 (4Ω) ═══► 8Ω total
│      │
│ Ch B ├────► Cab 3 (4Ω) ──jumper──► Cab 4 (4Ω) ═══► 8Ω total
└──────┘
```

**Per driver:** Voice coils in series (2Ω + 2Ω = 4Ω)  
**Per channel:** Two cabinets in series (4Ω + 4Ω = 8Ω)  
**Power:** ~500-600W per driver (excursion-limited at 15 Hz)

**Cable runs:**
- 2 runs from amp (one per channel)
- 2 short jumpers between cabinet pairs

**Terminal cups:** Dual binding post (input + output for series chain)

## Assembly Sequence

### Phase 1: Preparation
1. Laminate baffle layers (glue + clamp 24hrs)
2. Cut driver holes (router circle jig, Ø16.875")
3. Build window brace frames

### Phase 2: Box Assembly
4. Assemble box: Back → Sides → Top/Bottom
5. Install cleats (glue + screw, 2" from front)

### Phase 3: Internal Bracing
6. Install front-to-back window braces
7. Install side-to-side window brace

### Phase 4: Wiring & Damping
8. Install terminal cup (dual binding post)
9. Run speaker wire
10. Install damping material

### Phase 5: Baffle Installation
11. Apply gasket tape to cleat faces
12. Screw baffle into cleats
13. Mount driver
14. Connect speaker wire
