## Safety & Code Compliance

### Electrical

- All AC wiring by licensed electrician
- Dedicated 20A circuits for amplifiers (front stage equipment rack)
- Proper grounding throughout
- GFCI protection where required by code

### Structural

- Stage platform properly braced (no flex/bounce)
- Weight capacity: Minimum 100 lbs/sq ft (exceeds requirements)
- Riser platform properly braced (no flex/bounce)
- Professional structural review recommended for large riser

### Egress

- Verify room meets code for emergency exit
- Exit signage as required by jurisdiction
- Path to exit clearly defined (even if not illuminated during use)
- Consider secondary exit if room has only one door

### Fire Safety

- Acoustic treatment materials: Class A fire rating preferred
- Fabrics: FR (flame retardant) rated (Guilford of Maine offers FR options)
- Smoke detectors: Code-required placement
- Fire extinguisher: Accessible location

### Subwoofer Driver Protection

- **CRITICAL:** High-pass filtering at 18-20 Hz is MANDATORY
- Configure in miniDSP before any high-level playback
- Verify during calibration
- Without filter, extreme subsonic content could exceed safe excursion
- This is essential for equipment longevity and optimal performance

## Maintenance & Long-Term Considerations

### Subwoofer Maintenance

**Regular Checks:**
- Inspect driver surrounds for deterioration (annually)
- Verify enclosure seals (check for air leaks at driver mounting)
- Test binding posts for tightness
- Clean dust from driver cones (soft brush, no contact)

**Front subs (under stage):**
- Access via removable AT fabric panels on stage front
- Check cable connections in corner pockets
- Verify drivers firing properly through openings

**Rear subs (behind riser):**
- Access via removable AT fabric panels
- Standard maintenance procedures

**Amplifier Maintenance:**
- Ensure adequate ventilation (rack cooling)
- Check connections periodically
- Monitor for thermal shutdown (indicates ventilation issue)
- Equipment rack on stage provides excellent access

**DSP Maintenance:**
- Back up miniDSP configuration (save presets)
- Update firmware as released
- Maintain calibration files (Dirac measurements)

### Room Maintenance

**Acoustic Treatment:**
- Vacuum fabric panels (low suction)
- Inspect for sagging or damage
- Replace as needed (10-20 year lifespan typical)

**Stage Platform:**
- Check for squeaks or movement (indicates loose fasteners)
- Inspect Sorbothane pads for compression (replace every 5-10 years)
- Equipment rack: Verify stability, check connections

**Riser Platform:**
- Check for squeaks or movement (indicates loose fasteners)
- Inspect Sorbothane pads for compression (replace every 5-10 years)
- Verify couch positioning (ensure isolation pads remain under feet)

**Screen:**
- Clean AT fabric carefully (manufacturer instructions)
- Avoid touching (oils damage fabric)
- Inspect for tears or snags

### Technology Updates

**Future-Proofing Considerations:**
- HDMI infrastructure (conduit for future cable replacement)
- Modular amplification (can upgrade individual channels)
- Projector budget allows for best available in 2027
- Speaker system timeless (quality drivers don't become obsolete)

**Potential Upgrades (Post-2027):**
- Projector replacement (every 5-10 years as technology advances)
- Dirac software updates (new features, better algorithms)
- Additional Atmos channels if standards evolve (9.4.6 ×  9.4.8, etc.)
- Subwoofer amplifier upgrades (more power if desired)
