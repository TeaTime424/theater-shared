## Driver Cutout Details

### Cutout Dimensions
- Diameter: 16.875"
- Bolt circle: 17.75" diameter
- Bolt holes: 8× evenly spaced (45° apart)
- Bolt size: 1/4" (drill 9/32" for clearance)

### Center Point Location
- All driver holes centered in panel
- Center point: 13.25" from all edges (on 26.5" panel)

### Cutting Method
1. Mark center point
2. Use router with circle jig, OR
3. Drill pilot hole, cut with jigsaw, sand to line

## Terminal Plate Cutout

### Location: Back Panel (Lower Center)

The terminal plate mounts on the **back panel**, positioned low for easy wire routing and access.

```

│                         │  ↑

│      │  PLATE  │        │  │    centered horizontally
│      └─────────┘        │  ↓
└─────────────────────────┘
←───────── 25" ──────────→
```

### Terminal Plate Specifications

| Parameter | Value |
|-----------|-------|
| Plate type | Dual binding post (5-way) |
| Typical cutout | 3.5" W × 2.5" H (verify with actual plate) |
| Position (horizontal) | Centered (12.5" from either edge) |
| Position (vertical) | 5" from bottom edge of panel |
| Clearance | Above lower cross-brace (at 9.5"), below driver motor |

### Common Terminal Plate Sizes

| Style | Cutout Size | Notes |
|-------|-------------|-------|
| Standard dual binding post | 3.5" × 2.5" | Most common |
| Round cup style | 3" diameter | Alternative look |
| Large rectangular | 4.5" × 3" | Higher current capacity |

**Recommended:** Parts Express #260-303 or similar dual 5-way binding post plate.

### Cutting Method

1. Mark centerlines at 12.5" from side, 5" from bottom
2. Trace terminal plate outline (or use template)
3. Drill corner relief holes (1/4" bit)
4. Cut with jigsaw
5. File/sand edges smooth
6. Test fit before assembly

### Wiring Access

Terminal position provides clear internal wire routing:
- Top driver wire: runs down from motor, connects at terminal
- Bottom driver wire: runs up from motor, connects at terminal
- Both wires meet at terminal for series connection
