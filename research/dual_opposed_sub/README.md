# Dual-Opposed Vertical Subwoofer

## Overview

Force-canceling corner-loaded subwoofer design using two Dayton Audio UMII18-22 drivers in vertical opposed configuration (one facing up, one facing down).

**Key Features:**
- Dual-opposed drivers cancel cabinet reaction forces
- 29.5" cubic form factor optimized for corner placement
- +12 dB boundary gain from three-wall corner loading
- Same acoustic performance as front-firing design (109 dB @ 20Hz at MLP)
- Reduced cabinet vibration and "walking"
- **Doubled bottom panel (1.5") for secure inverted driver mounting**

## Why Double the Bottom Panel?

The bottom driver (47 lbs) hangs inverted from the panel. Unlike the top driver which compresses into its panel, the bottom driver creates tension on T-nuts and mounting bolts.

- **Top panel (0.75" single):** Driver weight pushes INTO panel — stable
- **Bottom panel (1.5" doubled):** Driver weight pulls AWAY — needs reinforcement

Doubling provides:
- Secure T-nut embedment depth
- Reduced panel flex during excursion
- Same approach used for vertical baffles in front-firing designs
- Peace of mind for 47 lbs hanging inverted for 20+ years

## Assembly Order Summary

1. Cut all MDF panels per layout diagrams
2. **Laminate bottom panels** (glue + clamp both 29.5" × 29.5" layers)
3. **Cure 24 hours minimum**
4. Cut 16.875" driver holes through both panels (top single, bottom doubled)
5. Drill bolt holes, install T-nuts (inside face)
6. Cut window braces (12" center window)
7. Assemble four sides into square frame (27.25" tall)
8. Install window braces at 9.5" from top/bottom
9. Attach bottom panel (1.5" doubled)
10. Attach top panel (0.75" single)
11. Attach corner legs (screws from inside into 1.5" panel)
12. Install terminal plate, seal all joints
13. Install bottom driver (face down, **2.5" bolts**)
14. Install top driver (face up, 2" bolts)
15. Install protective grille
