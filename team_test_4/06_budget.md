## Critical Assessment: Where the Money Goes

### Spend Distribution

| Category | Cost | % of Total |
|----------|------|------------|
| Building prep (Phase 1) | $22,559 | 46.3% |
| Projector + screen | $7,599 | 15.6% |
| Processor + DSP + Dirac | $4,697 | 9.6% |
| Sub system (drivers + amps + enclosures) | $5,565 | 11.4% |
| Speakers (Volt-10s) | $2,504 | 5.1% |
| Seating + construction | $4,040 | 8.3% |
| Infrastructure + shakers | $1,740 | 3.6% |
| **Total** | **$48,704** | **100%** |

Building prep is nearly half the budget -- this is the cost of converting a garage to a livable space. The theater-specific equipment (Phase 2) is $26,145.

### Biggest Savings Opportunities

1. **Projector downgrade** (NZ500 → NZ7 or Epson LS12000): Save $1,500-3,000 but lose native 4K and contrast
2. **Reduce sub count** (8 → 4): Save ~$2,100 but lose 3 dB headroom and mode smoothing
3. **Skip Dirac ART** ($299): Lose per-sub MIMO correction -- not recommended
4. **DIY seating**: Save ~$500-1,000 with used furniture
5. **Phase 2 staging**: Build in phases -- speakers first, subs second, treatment third

**Bottom line:** The design is already optimized. Major savings require performance sacrifices. The owned inventory ($13,450) provides significant leverage -- without it, out-of-pocket would be $62,154.
