# 12.5 cu ft Dual-Driver Subwoofer - BUILD GUIDE

**Configuration:** Sealed, 2× Dayton UMII18-22 stacked vertically  
**Bracing:** 5 identical U-braces with notched extended legs  
**Baffle Support:** Continuous perimeter cleats (pass through leg notches)  
**Materials:** 2× 4'×8' MDF sheets (0.75")  
**Joinery:** Glue + clamps + brad nails (no screws)

---

| Parameter | Value |
|-----------|-------|
| External | 30" W × 48" H × 21.5" D |
| Internal | 28.5" W × 46.5" H × 16.25" D |
| Volume | 12.47 cu ft gross |
| Drivers | 2× UMII18-22 @ Y=12.81" and 33.69" |
| Baffle | 1.5" laminated, 3" recess |
| U-Braces | 5 identical @ Y=2.2", 12.81", 23.25", 33.69", 44.3" |

## Joinery Method

**Glue is the primary structural bond.** Brad nails simply hold panels in position while glue cures. This approach eliminates the risk of MDF blowout from misaligned screw pilots.

```
JOINT ASSEMBLY SEQUENCE:
    
1. Dry fit - verify alignment and square
2. Apply Titebond III to mating surfaces
3. Position panels
4. CLAMP TIGHTLY - this creates the glue bond
5. Shoot brads while clamped - holds position
6. Leave clamps 30-60 min until glue tacks
7. Move to next joint
```

**Critical:** Brads alone don't pull joints tight — clamping does. Don't skip step 4.

### Cutting the Notch
```
METHOD 1: Jigsaw
    1. Mark ¾" × ¾" square on inside front corner
    2. Cut along both lines with jigsaw
    3. Clean up with chisel or file

METHOD 2: Table Saw (if available)
    1. Set blade height to ¾"
    2. Set fence to ¾" from blade
    3. Make two perpendicular cuts
    4. Piece falls away cleanly

METHOD 3: Handsaw + Chisel
    1. Mark lines with square
    2. Cut both lines with handsaw
    3. Chisel out the corner
```

## Brad Nailing Tips for MDF

```
18GA BRAD NAILING - BEST PRACTICES

✓  Shoot perpendicular to surface
✓  Space 6" apart on shell joints
✓  Space 4-5" apart on braces and cleats
✓  Keep 1" from panel edges/corners
✓  Adjust nailer depth — heads should be flush or slightly below
✓  No pre-drilling required

✗  Don't shoot at extreme angles (brads will curve)
✗  Don't rely on brads alone — always clamp first
✗  Don't shoot into unsupported edges
```
