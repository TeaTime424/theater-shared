## Wiring

### Driver Configuration (per cabinet)
- 8 drivers split into 2 groups of 4
- Each group: 2S2P configuration = 4Ω per group
- Cabinet presents: 2× 4Ω loads (one per amp channel)

### Wiring Diagram (per group of 4)
```
                 ┌─────────────────┐
                 │                 │
            ┌────┴────┐       ┌────┴────┐
            │ Driver  │       │ Driver  │
            │   4Ω    │       │   4Ω    │
            └────┬────┘       └────┬────┘
                 │ (series)        │
            ┌────┴────┐       ┌────┴────┐
            │ Driver  │       │ Driver  │
            │   4Ω    │       │   4Ω    │
            └────┬────┘       └────┬────┘
                 │                 │
                 └────────┬────────┘
                      (parallel)
                          │
                     GROUP = 4Ω
```

### Terminal Cups
- Quantity: 2 per cabinet (one per driver group)
- Type: Dual binding post or Speakon
- Location: Back panel, lower corners

## Construction Notes

### Assembly Order
1. Laminate baffle layers (glue + clamp 24hrs)
2. Cut all panels to size
3. Install cleats on sides/top/bottom
4. Attach back panel to sides
5. Add top and bottom
6. Install bracing
7. Cut driver holes in baffle
8. Install baffle on cleats
9. Wire drivers, install terminal cups
10. Mount drivers
11. Apply finish
12. Install feet

### Critical Tolerances
- Baffle flatness: ±1/16" over length
- Driver hole diameter: 9.125" +1/16" / -0"
- Panel squareness: check diagonals within 1/8"
- Seal all joints with wood glue + caulk

### Sealing
- All panel joints: wood glue + screws
- Internal edges: silicone caulk bead
- Terminal cups: foam gasket + silicone
- Driver gaskets: closed-cell foam tape
