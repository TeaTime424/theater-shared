# 8 cu ft Dual Front-Firing Subwoofer - Bracing Details

## Overview

The dual front-firing configuration requires enhanced bracing because:

1. **No force cancellation** — Both drivers push in the same direction simultaneously
2. **Tall cabinet** — 44" height creates longer unsupported panel spans
3. **High excursion** — 28mm Xmax creates significant internal pressure swings

---

## Bracing System Components

### 1. Cross-Braces (Interlocking)

**Purpose:** Stiffen side panels, resist pressure-induced flexing

**Configuration:** Two levels of interlocking + braces

```
TOP VIEW (cross-brace level):

    ┌─────────────────────────┐
    │                         │
    │    ┌─────────────┐      │
    │    │             │      │
    │────┤  INTERLOCK  ├──────│
    │    │             │      │
    │    └─────────────┘      │
    │                         │
    └─────────────────────────┘
         Back
```

**Level 1 Position:** 14" from bottom (below bottom driver motor)
**Level 2 Position:** 28" from bottom (above bottom driver, below top driver motor)

**Dimensions:**
| Part | Size | Qty |
|------|------|-----|
| Width Brace | 22.0" × 4.0" × 0.75" | 2 per level |
| Depth Brace | 12.25" × 4.0" × 0.75" | 2 per level |

**Notch Detail:**
```
           22.0"
    ├─────────────────────┤
    ┌─────────┬───┬───────┐
    │         │   │       │  4.0"
    │         │2.0│       │
    └─────────┴───┴───────┘
              ↑
         2.0" deep notch
```

Both braces have centered notches. When assembled, they interlock:

```
ASSEMBLED (top view):

              Depth brace
                  │
    ──────────────┼──────────────  Width brace
                  │
```

### 2. Rib Braces

**Purpose:** Break up panel resonances, prevent drumming

**Configuration:** Vertical strips on side panels and back panel

```
SIDE PANEL (interior view):

    ┌─────────────────────────┐
    │    ║              ║     │
    │    ║              ║     │
    │    ║              ║     │
    │    ║              ║     │
    │    ║              ║     │
    │    ║              ║     │
    │    ║              ║     │
    └─────────────────────────┘
         ↑              ↑
      7.5" from      7.5" from
      front edge     back edge
```

**Dimensions:**
| Part | Size | Qty |
|------|------|-----|
| Side Rib | 1.5" × 32.5" × 0.75" | 4 (2 per side) |
| Back Rib | 1.5" × 32.5" × 0.75" | 2 |

**Positioning:**
- Ribs run vertically
- Positioned to avoid cross-brace interference
- Start 5" from top/bottom to clear cross-braces

---

### 3. Corner Braces

**Purpose:** Reinforce plywood edge joints (critical for plywood construction)

**Configuration:** L-shaped braces in all 4 vertical corners

```
CORNER DETAIL (top view):

    Panel edge
        │
        │  ┌───┐
        │  │   │ 3"
        └──┤   │
           │   │
           └───┘
             3"
```

**Dimensions:**
| Part | Size | Qty |
|------|------|-----|
| Corner Brace | 3" × 3" L × 42.5" long | 4 |

**Options:**
1. **L-shaped:** Cut from 3" × 6" stock, rip lengthwise at 45°
2. **Flat triangle:** 3" × 3" right triangle profile
3. **Solid blocks:** 3" × 3" × 42.5" with corner removed

---

### 4. Baffle Stiffener

**Purpose:** Prevent baffle flex between driver cutouts

**Configuration:** Vertical block between drivers

```
FRONT BAFFLE (interior view):

    ┌─────────────────────────┐
    │                         │
    │      ┌───────────┐      │
    │      │  DRIVER   │      │
    │      └───────────┘      │
    │           ║             │  ← Stiffener (3" × ~7")
    │      ┌───────────┐      │
    │      │  DRIVER   │      │
    │      └───────────┘      │
    │                         │
    └─────────────────────────┘
```

**Dimensions:**
| Part | Size | Qty |
|------|------|-----|
| Baffle Stiffener | 3.0" × 7.0" × 3.0" | 1 |

**Note:** This stiffener bridges the gap between driver cutouts. It's glued to the back of the inset frame assembly, spanning from one driver opening to the other.

## Installation Sequence

1. **Rib braces first** — Attach to panels before cabinet assembly
2. **Corner braces during assembly** — Install as sides go together
3. **Cross-braces after frame** — Install before top/bottom panels
4. **Baffle stiffener** — Attach to inset frame before mounting to cabinet

---

## Volume Displacement

| Component | Displacement |
|-----------|-------------|
| Cross-braces (8 pcs) | 0.31 cu ft |
| Rib braces (6 pcs) | 0.17 cu ft |
| Corner braces (4 pcs) | 0.44 cu ft |
| Baffle stiffener | 0.04 cu ft |
| **Total Bracing** | **~0.96 cu ft** |

This is accounted for in the net volume calculation.
