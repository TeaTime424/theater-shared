# 8 cu ft Dual Front-Firing Subwoofer - Cut List

## Material: 3/4" Plywood (23/32" actual)

---

## Design Overview

The front baffle is a **double-layer (1.5" total)** panel mounted **3" back from the front edge** of the cabinet. The sides, top, and bottom panels extend forward to create a recessed "pocket" in front of the drivers.

```
TOP-DOWN VIEW:
                    
    ┌─────────────────────┐
    │                     │ ← Front opening (3" deep recess)
    │  recess             │
    ├─────────────────────┤ ← Baffle (1.5" thick, recessed)
    │                     │
    │   Sealed chamber    │
    │                     │
    └─────────────────────┘ ← Back panel
```

### Why Double-Layer Baffle?

- **1.5" thickness** provides rigidity for heavy 18" drivers (47 lbs each)
- Allows proper T-nut installation with adequate thread engagement
- Standard practice for subwoofer baffles with large drivers
- Both layers get matching driver cutouts, laminated together

## Panel Dimensions Explained

**External Cabinet:** 23.5" W × 18.5" D × 44.0" H

**Back Panel:** 23.5" × 44.0"
- Full external width × full external height

**Side Panels (×2):** 17.75" × 44.0"
- Depth: 18.5" - 0.75" (back) = 17.75"
- Full external height (44")

**Top/Bottom Panels:** 22.0" × 17.75"
- Width: 23.5" - 1.5" (two sides) = 22.0"
- Depth: Same as sides = 17.75"

**Front Baffle (×2 layers):** 22.0" × 42.5"
- Width: Fits between sides = 22.0"
- Height: 44.0" - 1.5" (top+bottom) = 42.5"
- **Two layers laminated = 1.5" total thickness**

## Baffle Lamination Notes

1. Cut driver holes in BOTH baffle layers
2. Drill bolt holes through BOTH layers (use template)
3. Dry-fit to verify alignment
4. Apply wood glue between layers
5. Clamp with cauls for flat, even pressure
6. Use temporary bolts through driver holes to maintain alignment
7. Let cure 24 hours before handling
8. Install T-nuts from back (inside) face

## Assembly Order

1. Cut all panels from sheets
2. Cut driver holes in BOTH baffle layers
3. Drill bolt holes in BOTH baffle layers
4. Laminate baffle layers together (glue + clamp 24hrs)
5. Install T-nuts in laminated baffle (from back)
6. Seal interior surfaces
7. Attach rib braces to sides/back
8. Assemble back + sides
9. Install cross-braces at 14" and 28"
10. Attach bottom, then top
11. Install cleats (4.5" from front edge)
12. Install baffle on cleats
13. Seal all joints with caulk
14. Finish exterior
15. Install terminal plate and drivers
