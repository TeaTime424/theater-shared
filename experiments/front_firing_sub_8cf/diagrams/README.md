Download these PNG files from the Claude outputs interface.

|------|-------------|

| ff_02_panel_dimensions.png | Individual panel dimensions and details |
| ff_03_cross_section_front.png | Front cross-section showing drivers and bracing |
| ff_04_cross_section_side.png | Side cross-section showing depth and inset |
| ff_05_cross_section_top.png | Top cross-section showing bracing layout |
| ff_06_3d_view.png | 3D isometric view with specifications |
| ff_07_ortho_views.png | Orthographic views (front, side, top, back) |
| ff_08_step_panels.png | Build Step 1: Cut all panels |
| ff_09_step_driver_holes.png | Build Step 2: Cut driver holes |
| ff_10_step_laminate.png | Build Step 3: Laminate inset frame |
| ff_11_step_assemble.png | Build Steps 4-6: Assemble cabinet frame |
| ff_12_step_baffle.png | Build Step 7: Attach front baffle assembly |
| ff_13_step_drivers.png | Build Step 8: Mount drivers and wire |

## Download Instructions

1. In the Claude interface, scroll to find the PNG files in the outputs
2. Click each file to download
3. Save to this `diagrams` folder

## Regenerating Diagrams

The Python scripts in the `intermediates` folder can regenerate these diagrams.
See `intermediates/README.md` for instructions.
